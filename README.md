# Customer Segmentation using K-Means Clustering and Power BI

## Project Overview
This project segments customers based on demographic information and purchasing behavior using the K-Means clustering algorithm. The results are visualized in an interactive Power BI dashboard to support business decision-making.

## Objectives
- Perform customer segmentation using K-Means clustering.
- Analyze customer purchasing patterns.
- Visualize customer segments in Power BI.
- Generate business insights and recommendations.

## Tools Used
- Python
- Google Colab
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Power BI

## Dataset
Mall Customers Dataset

Features:
- CustomerID
- Gender
- Age
- Annual Income (k$)
- Spending Score (1-100)

## Files Included
- Customer_Segmentation.ipynb
- Customer_Segmentation.pbix
- Mall_Customers.csv
- Customer_Segmentation_Result.csv

## Project Outcome
The project successfully identified customer segments based on demographic and spending behavior. These insights can help businesses create targeted marketing strategies and improve customer retention.
